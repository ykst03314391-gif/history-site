# 類似サイト調査と差別化検討（2026-07-08 時点）

戦国クロニクルと「似たこと」をしているサイトを類型別に把握し、差別化の材料にする。
※ 各サイトの評価は調査時点の概観。実際の運用前に再確認する。

## 類型別の主なプレイヤー

### A. 事典・データベース系（網羅で勝負）
- **刀剣ワールド／ホームメイト**（touken-world.jp）… 武将年表・合戦・刀剣・イラストの巨大DB。網羅性・SEOが非常に強い。
- **メイハク（名古屋刀剣ワールド）**（meihaku.jp）… 戦国武将一覧。
- **ジャパンナレッジ**（有料）／**Wikipedia** … 事典。
- 雑誌系：デアゴスティーニ『週刊 戦国武将データファイル』。
- → **網羅・中立の事典。ここと網羅性で戦うのは不利。**

### B. 歴史Webメディア（記事で勝負）
- **戦国ヒストリー**（sengoku-his.com）… 本格派。**複数ライター**が専門書・自治体情報をもとに丁寧に執筆。品質が高い。
- **武将ジャパン／BUSHOO!JAPAN**（bushoojapan.com）… 「日本初の歴史戦国ポータル」。大河解説・記事量が多い。
- **レキシル**（rekisiru.com）… 「歴史＝エンタメ」。読みやすさ・初心者向け。
- **戦国BANASHI**（sengokubanashi.net／YouTubeも）… 大河・観光も。
- → **記事メディアの層は厚い。ただし多くは「複数ライターの解説」で、一貫した“個人の視点”は薄い。**

### C. 城・史跡めぐり（現地で勝負）
- **攻城団**（kojodan.jp）… 4,563城のDB＋**訪問記録**＋**御城印13,200**＋城メモ（見どころ）＋城たび（旅）。城分野は圧倒的。
- **城びと**（shirobito.jp）… 城の**最新研究・企画展情報**も扱う。
- ニッポン城めぐり（位置ゲー）等。
- → **城の記録・網羅は攻城団が最強。ここも網羅では勝てない。**

### D. お出かけ・展示情報（アグリゲーター）
- **インターネットミュージアム**（museum.or.jp）… 全国1,100件超の考古・歴史展示を網羅。
- **Tokyo Art Beat／いこーよ／アイエム** … 展示・お出かけの一般アグリゲーター。
- → **展示情報の網羅は既存の一般ミュージアム系が強い。ただし“戦国特化”でも“見立て付き”でもない。**

### E. 動画（アテンションの主戦場）
- **戦国BANASHI（Mr.武士道）／YUKIMURAチャンネル（15.9万人）／よしふじ戦国／合戦歴史／歴史ゆっくり紀行** … 地図付き合戦解説、ゆっくり解説など。
- → **“わかりやすい解説”は動画が強い。テキスト×出典×視点で棲み分ける。**

### F. 個人ブログ・note
- にほんブログ村（戦国ランキング）、note（戦国パンダ 等）… 個人の考察・想像。玉石混交。
- → **“一人の視点”という点は近い。ただし出典の裏取りや設計は弱いことが多い。**

## 各類型に対する戦国クロニクルの立ち位置

| 類型 | 王者 | 網羅で勝てるか | 我々の差別化 |
|---|---|---|---|
| 事典・DB | 刀剣ワールド | ✕ | DBは「考察の脚注」。網羅せず、考察が要る範囲だけ |
| 記事メディア | 戦国ヒストリー | △（質で拮抗は困難） | **一人の一貫した視点／中立を装わず自説を出す** |
| 城・史跡 | 攻城団 | ✕ | 記録ではなく**解釈**。考察→現地の動線。攻城団とは共存（リンクも可） |
| 展示・お出かけ | インターネットミュージアム | ✕ | **戦国特化＋管理人の見立て＋人物DB連携**で厳選 |
| 動画 | YouTuber群 | ✕（媒体が違う） | テキスト＋出典＋深い考察。将来は動画連携も |
| 個人ブログ | － | － | 出典の裏取り・設計・データ連携で差 |

## 差別化の結論（材料）

**1. どのカテゴリも強い専業がいる。単機能では勝てない。**
→ 網羅（DB・城・展示）でも、わかりやすさ（動画）でも正面から戦わない。

**2. 戦国クロニクルの武器は「束ねる視点」。**
- **一人の一貫した視点**（＝背骨）。戦国ヒストリー等の“複数ライターの事典的記事”にない、署名性・態度。
- **原典での裏取り＝誠実さ**（信長公記など reference/ で検証）。個人ブログにない厳密さ。
- **考察 ⇄ 現地（お出かけ）⇄ 合戦史跡 ⇄ ゲーム ⇄ 新説** を**同じ視点で統合**。単機能サイトにない横断。
- 親しみ（ぱぱねこ）でエンタメ層にも接近しつつ、出典で差。

**3. 具体アクション候補**
- **相互リンクの統合を強化**：「この考察の現地はここ／この合戦の史跡はここ／今その県で見られる展示はこれ」を一気通貫に。**これは我々だけができる束ね方**。
- **共存戦略**：城の記録は攻城団、展示網羅はインターネットミュージアムへ**リンクしてよい**（役割が違うと明示）。我々は“解釈と動線”を担う。
- **お出かけは網羅を捨てて厳選＋見立て**（戦国特化・人物連携）。
- **新説×ゲーム×お出かけを「同じ態度」で回す**のは希少。ここを前面に。

## 追補：deep-research による深掘り（2026-07-09）

> **検証ステータスの注記**：自動リサーチの「敵対的検証」フェーズがセッション上限で失敗したため、以下は**取得元ページから抽出した一次情報（出典URL付き）だが、三重検証は未了**。定性的な positioning は信頼できるが、**定量値（登録者数・PV・支援人数・設立日等）は要再確認**。

### ★ 最重要の発見：ほぼ直接の競合が既に存在する

- **戦国ウォーカー（sengoku-walker.blog.jp）** … **管理人一人**が運営する戦国情報ウェブマガジン。**特別展・博物館レビュー／大河・ドラマ評／史跡／ゲーム／新説／ニュース集約**を270超カテゴリで**横断**。収益は Amazonアソシエイト＋広告＋寄付。
  → **「お出かけ×史跡×ゲーム×新説を一人で横断統合」は既に先行例がある。** ただし内容は**二次情報中心**（"展示で見た/ドラマで描かれた"）で、**一次史料の直接引用・史料批判・一貫した自説考察は手薄**。
- **武将ジャパン**は既に**信長公記の全巻通読連載**（本郷和人の分析コラム）を行い、**一次史料検証**を明示。ただし**多人数編集**。

**→ 差別化の再定義（重要）：**
「横断統合」**だけ**では戦国ウォーカーに先行され、「原典裏取り」**だけ**では武将ジャパンに部分占有されている。
戦国クロニクルの本当の空白＝**「一人の一貫した署名視点 × 一次史料での裏取り × 横断統合」の三点セット**。三つの“掛け算”が誰も埋めていない領域。

### 運営体制の実態（＝“一人の視点”の希少性の裏づけ）

| サイト | 運営体 | 視点 |
|---|---|---|
| 刀剣ワールド／メイハク | **東建コーポレーション（不動産大手）系の財団**（匿名・委員会編集） | 一人の視点は皆無 |
| 戦国ヒストリー | 編集メディア（渡邊大門・濱田浩一郎ら**複数の署名ライター**） | 多視点 |
| 武将ジャパン | 編集運営（編集管理人＋執筆陣＋本郷和人連載） | 多視点 |
| レキシル | 株式会社Intro（**エンタメ志向・汎歴史**） | 企業編集 |
| 攻城団／城びと | 法人（UGC＝ユーザー投稿型DB） | 視点より記録 |
| **戦国ウォーカー** | **個人** | 一人（ただし裏取り弱い） |

→ 大手はほぼ**法人・匿名・多人数**。「**一人の署名された一貫視点**」自体が希少資源。

### 収益化の実例（＝アフィリ単体では弱い。多角化が定石）

| 事例 | 収益モデル |
|---|---|
| 攻城団 | **個人サポーター月額課金**（約591人・目標1,000）＋企業スポンサー＋広告＋物販（御城印帳・トレカ）＋自治体の観光マーケ支援。※月間約88万PV |
| 戦国BANASHI | 広告＋**有料メンバーシップ**（ニコニコ FC 月1,000円／3,000円）。登録者約21万 |
| 城びと | EC（城びとストア）＋会員＋広告＋自治体タイアップ＋お城EXPO |
| 刀剣ワールド財団 | **物販＋出版**（楽天は自社“通販”でアフィリではない） |
| 戦国ウォーカー | Amazonアソシエイト＋広告＋寄付（個人規模） |

→ 我々の方針（アフィリ＋**note有料/メンバーシップ**＋物販）は競合実例に裏づけられる。とくに**会員課金**は個人メディアの安定収益の実証例（攻城団・戦国BANASHI）。

### 差別化戦略（改訂）
- **勝てる（空白）**：一人の署名視点 × **一次史料の裏取り** × 横断統合、の**三点セット**。とくに戦国ウォーカーとの差＝**裏取り・考察の質**、武将ジャパンとの差＝**単著の一貫性**。
- **戦わない**：網羅DB（刀剣ワールド財団・攻城団）／展示網羅（インターネットミュージアム）／動画のわかりやすさ（戦国BANASHI等）。
- **共存・参照**：城データは攻城団、展示網羅はインターネットミュージアムへリンク。
- **収益**：アフィリ単体に頼らず、早期から**会員課金/note有料＋物販（ぱぱねこ）**を併走。

## 追補：別軸の検討「日本史 × 経済 / 保険・リスク」（2026-07-10）

戦国特化の枠を超え、**「日本史 × 経済」「保険/ライフプランニング的視点」**を主眼にできないか、を検討した際の競合調査。結論と根拠を残す。

### 軸1：日本史 × 経済 ― 書籍はレッドオーシャン、Web個人メディアは手薄

- **書籍は巨匠が全時代を制覇済み**＝コンセプト単体では新規性ほぼゼロ。
  - 上念司『経済で読み解く日本史』全6巻（縄文〜昭和を時代別。室町・戦国編はデフレ経済で応仁の乱・戦国突入を説明）。
  - 大村大次郎『お金の流れで読む日本の歴史』（元国税調査官・著作30冊超。「信長は金融改革でデフレ解消」等）。
  - 飯田泰之『日本史で学ぶ「貨幣と経済」』『日本史に学ぶマネーの論理』（経済学者・厳密系）、横山和輝『日本史で学ぶ経済学』、井沢元彦『お金の日本史』ほか。
- **Web/個人メディアは手薄**：検索の前面は書籍・金融機関の啓蒙ページ・Wikipedia・学術書ばかり。体系的な無料Web連載やnoteは目立たない。
  - ただし**「手薄＝空白」と即断は危険**。この題材は「通史でまとまって読みたい」需要が強いから本が売れる＝Web断片記事だと本の劣化版になりやすい。
- **勝ち筋**：背骨（一人の署名視点 × **一次史料の裏取り** × 横断統合）との掛け算。上念・大村は新書で一次史料の直接引用・史料批判は基本しない。「一次史料で裏を取りながら経済で読む／断定を避け諸説を明示する無料Web個人メディア」は空いている可能性が高い。

### 軸2：保険 / ライフプランニング的視点 ― (A)は避け、(B)＋翻訳法で行く

- **(A)「歴史人物・事件を現代の保険/FP視点で読む」当てはめ型 → 看板にはしない。**
  - 先行は保険会社・FP会社の**オウンドメディア**（マネードクター、三井住友海上、アフラック、かづな先生の保険ゼミ…）＝販売/集客SEOで浅い。「戦国武将×経営/マーケティング」も手垢つき。
  - 決定的弱点＝**アナクロ（時代錯誤）**。FP資格は日本で1987年創立と新しく、そのまま過去に投影すると「史実の考察」でなく「現代概念の投影」になり、背骨（一次史料裏取り）と相性が悪い。
  - → B層（お知らせ/フック）の単発ネタなら可。A層（看板）には据えない。
- **(B)「相互扶助・リスク分散の歴史そのものを掘る」保険前史型 → 筋が良い（本命）。**
  - 保険/共済の源流は**史実として実在**：義倉（奈良・平安）／頼母子講（初出=鎌倉1275年『高野山文書』）／無尽（室町以降）。「保険共済思想の源流」という研究もある。
  - **一次史料や研究で裏が取れる＝背骨と完全整合**。かつ軸1（経済×一次史料）と地続き。
  - 競合構図：保険会社オウンドメディアは"保険の歴史"を薄くまとめ、学術書は硬い。**その中間＝史実として深く・一次史料で・一人の視点で掘る個人メディアは空白**。
  - 具体の芽：徳政令＝借金帳消しの経済学（正長・嘉吉の土一揆）／講・無尽・義倉＝戦国のリスク分散、など。

### この2軸に対する方針（結論）

1. **看板の全面架け替えはしない**：戦国特化・DB・既存ポジションを薄めるリスク大。全日本史は一人だと浅くなる。
2. **経済を"日本史全体に射程を持つ縦串"として育てる**：戦国を主戦場に残し、経済シリーズ（`sengoku-economics` ほか）を通史的に拡張。必要に応じ平城京遷都・応仁・参勤交代へ射程を伸ばす。
3. **副軸に「リスク管理・相互扶助（保険前史）」を足す**：(A)投影は避け、(B)＋**現代フレームの翻訳的適用**（→ `method-modern-frames.md`）で書く。
4. **固有資産の活用**：管理人に保険/FPの実務知見があれば、それは上念（マクロ経済）・大村（税務）が持たない差別化源。ただし「現代知識の投影」でなく「史実のリスク管理を実務者の目で読み解く」方向で背骨と両立させる。
