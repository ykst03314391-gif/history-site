# 方法論：現代フレームの「翻訳的適用」

戦国クロニクル（および将来の日本史×経済/リスク）で用いる、記事の書き方の芯となる方法論。
2026-07-10 に確定。背骨（[[site-backbone]]・README「コンセプト」）の「捉え直す」を、具体的な技法に落としたもの。

## 何をする方法か

**現代の実務フレーム（リスク管理・ライフプランニング・保険・会計・マーケティング・兵站…）を、
過去の"行動の実態"に当てはめ、現代の語彙・ロジックで翻訳して理解可能にする。**

前提となる考え：
- 昔の人も**同じ人間**。備え・家・命をめぐる営み（実態）は大なり小なり行っていた。
- 無かったのは営みそのものではなく、**それを名づける言葉・枠組み**。
- だから現代の枠を「翻訳装置」として貸してやれば、現代人が地図を持って過去を歩ける。

## 投影（NG）と翻訳（OK）の一線

これは「面白い当てはめネタ」と紙一重で、踏み外すと俗説・広告記事に堕ちる。**次の区別が生命線**。

| | 投影（❌ やってはいけない） | 翻訳（⭕ これをやる） |
|---|---|---|
| 主張 | 「戦国武将は生命保険に入っていた」「FP的に設計していた」 | 「彼らの備え（講・分散相続・縁組・人質）は、現代語で言えばリスク分散だ」 |
| 中身 | 当時なかった**制度・概念を持っていた**と主張。史実を枠に合わせて歪める | 実在した**行動の実態**を、現代の語彙で言い換えて説明する |
| 記述の言語 | 現代の枠が**事実を供給**してしまう | 現代の枠は**記述・整理**に使い、事実は史料が供給する |

## 3つの歯止め（守れば投影に堕ちない）

1. **推論の向きを固定する**：`史料の実態 → 現代フレームで整理`。**データが先、枠は後。**
   枠を決めてから「合う史実」を探す/でっち上げる、をやった瞬間に俗説になる。
2. **フレームは"私（現代人）のレンズ"だと明示する**：「当時の人がそう考えていた」とは書かない。
   「私はこう捉える」と自分の視点として出す（＝背骨の「中立を装わず自説を明示」と一致）。
3. **"枠に合わない残余"を正直に書く**：現代フレームで見て**説明できない・ズレる部分**を必ず添える。
   そこに当時と現代の価値観の差が現れ、「投影」が「分析」に変わる。
   （関連テーマ：テーマ帳の「当時と現代の価値観の違い」）

## なぜ差別化になるか

- 既存の強者は**単一フレーム**：上念司＝マクロ経済、大村大次郎＝税務・お金の流れ（→ `competitors.md`）。
- 本サイトは**現代の実務フレーム"群"を、一人が一貫して過去の実態に当てる**。
  **フレームの束を翻訳装置として使う「方法」そのものが看板**になり、単発のネタ当てはめとは別物になる。
- 「経済」より人間の普遍的営み（備え・家・命）に触る分、**現代人の自分ごと化が効く**。

## 使いどころ（当面）

- **A層（看板の考察）**：翻訳法＋一次史料の裏取りで書く（例：講・無尽・徳政令＝リスク分散/相互扶助の実態）。
- **B層（お知らせ/フック）**：軽い当てはめネタは入口として可。ただし上の3歯止めを外さない。
- 経済シリーズ（`data/articles/sengoku-economics` ほか）の**副軸**として「リスク管理・相互扶助」を育てる。
  戦国を主戦場に残したまま、必要に応じて日本史全体へ射程を伸ばす（competitors.md の結論と整合）。
