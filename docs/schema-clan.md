# 大名家（clan）スキーマ設計 v0.1

「織田家」「上杉家」など、大名家を 1 家 1 ファイルで扱う。
`data/clans/<id>.json` に格納し、一覧は `data/clans/index.json` に持つ。

## 目的

- 「大名家から探す」機能の実体。家を選ぶ → その家の説明と所属人物を見る。
- 人物（person）と相互参照する（`members` の `person_id`）。
- 将来、記事の「大名家別の事例」（`relatedClan`）とも連携する。

## 一覧（index.json の各要素）

| フィールド | 型 | 説明 |
|---|---|---|
| `id` | string | 一意なID（例：`oda`） |
| `name` | string | 家名（例：`織田家`） |
| `kana` | string | 読みがな |
| `summary` | string | 一覧カード用の一言紹介 |
| `kamon` | string | 家紋名 |
| `region` | string | 主な勢力圏（例：`尾張・美濃`） |

## 詳細（<id>.json）

| フィールド | 型 | 説明 |
|---|---|---|
| `id` | string | 一覧と対応 |
| `name` | string | 家名 |
| `kana` | string | 読みがな |
| `summary` | string | 一言紹介 |
| `kamon` | string | 家紋名 |
| `base` | Place | 本拠・勢力圏（`name` / `prefecture`） |
| `head` | { person_id?, name } | 一門の基準となる主たる人物（当主）。一門の続柄はこの人から見た関係で書く |
| `description` | string[] | 説明本文（段落の配列。1要素＝1段落） |
| `members` | Member[] | 所属する人物（当主・後継・家臣など） |
| `tags` | string[] | タグ |
| `updatedAt` | string | 更新日 |

### Member（所属人物）
```
{ "person_id"?: string,   // 人物ページがあればリンク
  "name": string,         // 表示名（ページが無い場合も）
  "role"?: string,        // 家中での立場（例：当主 / 嫡男 / 重臣）
  "type"?: "一門" | "家臣" | "その他" }   // 分類
```
> **一門**：当主・血縁・一族。`role` には **`head` の人物から見た続柄**（父・弟・甥・嫡男・正室 など）を書く。
> **家臣**：その家に仕えた人物。`role` には立場（重臣・筆頭家老 など）を書く。
> **その他**：親交のあった公家・僧など、血縁でも家臣でもないゆかりの人物。
>
> 表示は「一門（◯◯から見た関係）」セクションと「ゆかりの人物」セクション（内部を家臣／その他に分ける）に分かれる。
> `type` 未設定は「一門」として扱う。

#### 一門の範囲（判断基準）

判断軸は「**その家に入るか／家から出るか**」。

**一門に含める**
- 当主の血族（父母・兄弟姉妹・子・孫・甥姪 など）
- 家に入った近親：**正室**、**迎えた養子**、**婿養子（娘婿）**
- **他家へ養子に出た実子**（血がつながるため、送り出した家の一門として記し、`role` に「◯◯家を継ぐ」と注記する）

**一門に含めない**
- **他家へ嫁いだ実娘**（婚姻により嫁ぎ先の家人となる。姻戚であって一門ではない）
- 血縁も姻戚もない家臣（→ `家臣`）

> ポイント：**娘婿・養子は「家に入る」ため一門**。**実娘を嫁がせるのは「家から出る」ため一門としない**。
> 他家へ嫁いだ娘は、その人物ページ（人物データの `family`）には残るが、実家の大名家ページの一門には載せない。

## 表示仕様

- 一覧：`clans.html` … `index.json` からカード表示（家を選ぶ）
- 詳細：`clan.html?id=<id>` … 説明本文＋所属人物（人物ページへリンク）

## 今後の拡張予定

- 家系図（`members` の関係を可視化）
- 大名家の年表（家の興亡）
- 記事の「大名家別の事例」を `relatedClan` で紐付け、この家ページに一覧表示する
- 人物データ側にも `clan` を持たせ、人物ページから家ページへ誘導する
