# reference/cinii/ ― CiNii 由来

[CiNii Research](https://cir.nii.ac.jp/) で確認した**論文の書誌情報メモ**を置く（全文ではなく、著者・誌名・巻号・年・crid/DOI 等の引用情報）。

- CiNii は書誌が確実に取れるが本文は取れないことが多い（PDFは機関リポジトリ側）。
- 現状：個別の書誌は記事側（`references`）に直接記載しているため、ここは空。まとまった書誌リストを作るときに使う。
- 例：藤城徹「本郷歴史サロン(58)『手取川の戦い』をめぐる議論」／川名俊「能登畠山氏の権力編成と遊佐氏」など。
