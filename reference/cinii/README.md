# reference/cinii/ ― CiNii 由来（論文の書誌メモ）

[CiNii Research](https://cir.nii.ac.jp/) で確認した**論文の書誌情報メモ**を置く（全文ではなく、著者・誌名・巻号・年・crid/DOI 等の引用情報）。CiNii は書誌が確実に取れるが本文は取れないことが多い（PDFは機関リポジトリ側）。

## 置き方：**発行年でサブフォルダ分け**

論文系は**投稿年（発行年）ごと**にフォルダを切る。1論文＝1ファイル（Markdown）。

```
cinii/
├─ 2025/
│  └─ fujishiro-tedorigawa.md
└─ 2021/
   └─ kawana-noto-hatakeyama.md
```
