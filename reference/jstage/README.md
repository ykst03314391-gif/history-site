# reference/jstage/ ― J-STAGE 由来

[J-STAGE](https://www.jstage.jst.go.jp/) のオープンアクセス論文で、**HTML本文が取得できたもの**を置く。

## 置き方：**発行年でサブフォルダ分け**

論文系は**投稿年（発行年）ごと**にフォルダを切る。1論文＝1ファイル。

```
jstage/
└─ <年>/        （例：2024/）
   └─ <著者-短題>.md  （本文が取れたら .txt も）
```

- 現状：**なし**。戦国史のテーマは J-STAGE に乏しく（「手取川」等で検索しても河川工学が中心）、歴史論文は機関リポジトリ／CiNii の方が有効。
- 使うときは来歴（著者・誌名・巻号・年・DOI/URL・取得日）を付ける。
