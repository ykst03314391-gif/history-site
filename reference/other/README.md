# reference/other/ ― その他の出典元

Wikisource / NDL / J-STAGE / CiNii 以外の由来をここに置く。

- 機関リポジトリ（大学紀要の OA 論文 PDF 等。PDFは全文抽出が不安定なので書誌＋要点メモ中心）。
- 青空文庫（戦国の一次史料は無いが、近代の歴史小説・随筆を「近代人がどう描いたか」の素材として使う場合）。
- 市販の刊本・現代語訳の参照メモ（フロイス日本史の完訳本など、ネットで全文が取れないもの）。

## 置き方

- **論文（機関リポジトリ等）は発行年でサブフォルダ分け**（`other/<年>/<著者-短題>.md`）。
- それ以外（刊本メモ・青空文庫 等）は種別で分けてよい（`other/books/` など）。

現状：**なし**。必要になったら来歴付きで追加する。
